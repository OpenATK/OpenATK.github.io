Trello App

Description


The OpenATK trello App allows the user to specify which OpenATK apps will synchronize with Trello. The app is responsible for retrieving and providing Trello authentication information to any other app which requests it through the use of a content provider. For a farm without a pre-existing Trello organization, the Trello App assists the user with setting up the farm organization and selecting members.
