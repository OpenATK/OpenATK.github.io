Adding operation type

To add a new operation type, start by tapping the drop-down navigation for operation type.


Then select "New Operation" at the bottom of the list.


Give your new operation type a name, maybe Fall Chisel 2012. Then tap "Add".


Your new operation type will appear in the drop-down, and you'll see all your fields in gray. You are ready to make operation records for the new operation type.
